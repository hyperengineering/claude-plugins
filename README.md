# Hyper Engineering Claude Plugins

Official plugin marketplace for Claude Code plugins from Hyper Engineering.

## Installation

Add this marketplace to Claude Code:

```
/plugin marketplace add hyperengineering/claude-plugins
```

## Available Plugins

| Plugin | Description | Install |
|--------|-------------|---------|
| recall | Experiential memory system for AI agents | `/plugin install recall@hyperengineering` |

## Plugin Details

### Recall

Recall provides experiential memory for AI coding agents, enabling them to:
- Record development insights, patterns, and decisions
- Query relevant knowledge using semantic search
- Learn from feedback to improve future suggestions

**Source:** [hyperengineering/recall-claude-plugin](https://github.com/hyperengineering/recall-claude-plugin)
